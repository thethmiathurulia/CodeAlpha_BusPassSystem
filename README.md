# City Bus Pass Management System
An AI-powered Cloud-based Bus Pass Application System developed as
part of the **CodeAlpha Cloud Computing Internship**.
## Features
- **Cloud Hosting:** Deployed on **AWS EC2** (Ubuntu Instance).
- **Web Server:** Powered by **Apache2**.
- **Backend:** Developed using **PHP**.
- **Database:** **MySQL** for persistent data storage.
- **Security:** Managed via AWS Security Groups (HTTP/SSH).
## Tech Stack
- Infrastructure: AWS EC2
- Server: Apache

- Languages: PHP, HTML, CSS
- Database: MySQL

## Setup Instructions
1. Clone the repository.
2. Import `database.sql` to your MySQL server.
3. Place `index.php` in your web root directory.
4. Update database credentials in `index.php`.
